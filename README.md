## 설치

### @cos-ui/react

#### npm

```sh
npm install @cos-ui/react
```

#### yarn

```sh
yarn install @cos-ui/react
```

### @cos-ui/primitives

#### npm

```sh
npm install @cos-ui/primitives
```

#### yarn

```sh
yarn install @cos-ui/primitives
```

### 기여를 위해 패키지를 받아 개발 시

- 패키지안에 있는 컴포넌트를 개발, 테스트하는 도구로 Storybook 을 사용하고 있습니다.

#### 의존성 설치

```sh
yarn install
```

#### 스토리북 실행

```sh
yarn storybook
```
